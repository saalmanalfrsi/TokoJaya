
# TokoJaya

## Deskripsi

TokoJaya adalah aplikasi desktop berbasis Java yang dikembangkan menggunakan NetBeans untuk membantu pengelolaan data pada toko. Sistem ini dirancang untuk mempermudah proses pencatatan, pengelolaan, dan penyimpanan data secara terkomputerisasi.

## Fitur

* Manajemen data
* Input dan pengolahan data
* Penyimpanan data ke database
* Antarmuka pengguna berbasis Java Swing
* Laporan data

## Teknologi yang Digunakan

* Java
* NetBeans IDE
* Java Swing
* MySQL
* JDBC

## Struktur Project

* `src/` : Source code aplikasi
* `nbproject/` : Konfigurasi project NetBeans
* `lib/` : Library yang digunakan
* `build/` : Hasil kompilasi
* `dist/` : File hasil build aplikasi
* `database_migration.sql` : Script database

## Cara Menjalankan

1. Clone repository ini.
2. Buka project menggunakan NetBeans IDE.
3. Import database menggunakan file `database_migration.sql`.
4. Sesuaikan konfigurasi koneksi database.
5. Jalankan project melalui NetBeans.
